Cola FIFO (FCFS)

Instrucciones:

El algoritmo FCFS (First-Come, First-Served) se basa en una cola estrictamente FIFO. Cuando la cola no está vacía y llega un nuevo proceso, debemos enlazar el actual último nodo (final) con el nuevo nodo para mantener la cadena.

AlgoritmoFCFS.c (Incompleto) 
// Función para encolar un proceso a la cola
void encolar(Cola* cola, Proceso proceso) {
    Nodo* nodo = crearNodo(proceso);
    if (cola->frente == NULL) {
        cola->frente = nodo;
        cola->final = nodo;
    } else {
        // [DESAFÍO] Enlaza el nodo actual 'final' con el nuevo 'nodo'
        // Pista: Accede al puntero 'siguiente' del nodo final
        cola->final = nodo;
    }
}
¡Correcto! cola->final->siguiente = nodo; crea el enlace físico en la lista enlazada, asegurando que el nuevo proceso quede formado detrás del último. Luego, se actualiza el puntero final.
Incorrecto. Debes asignar el nuevo nodo al puntero siguiente del nodo que actualmente está al final de la cola.
Estructura: cola->final->siguiente = nodo;
