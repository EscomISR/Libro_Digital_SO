Automatización de Tareas (Backup)

Instrucciones:

Un administrador de sistemas necesita automatizar el respaldo de archivos. Completa el script .bat utilizando los comandos correctos para crear directorios, copiar archivos y verificar la operación.

Respaldo.bat 
@echo off
echo Iniciando proceso de respaldo...

:: 1. Crear un directorio llamado 'Backups'
:: Pista: Usa el comando para 'make directory'
>> 
:: 2. Copiar todos los archivos de texto (.txt) a la carpeta 'Backups'
:: Pista: Comando para copiar archivos
>> 
:: 3. Verificar listando el contenido de 'Backups'
:: Pista: Comando para ver directorio
>> 
echo Respaldo completado con éxito.
pause
¡Excelente! Has usado correctamente los comandos básicos: mkdir para crear carpetas, copy para transferir archivos y dir para listar el contenido.
Inténtalo de nuevo. Revisa los comandos:
  • 1. mkdir o md
  • 2. copy o xcopy
  • 3. dir
